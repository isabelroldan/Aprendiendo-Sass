    escalar

    @function escalar($valor, $limite, $disminucion) { ... }

    Description

    Funcion para escalar una imagen con un bucle hasta un limite.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $valor

    Tamaño de la imagen

    		Number— none
    $limite

    Tamaño límite de la imagen

    		Number— none
    $disminucion

    De cuanto en cuanto disminuirá

    		Number— none

    Returns

    Number —

    '$valor' - El tamaño que debe tener la imagen y que ya haya superado el límite inferior.

    @mixin image($size, $avatar) { ... }

    Description

    Mixin para usar de manera recurrente y dar estilo a distintas imágenes

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Tamaño de la imagen

    		Number— none
    $avatar

    Tipo de la imagen. Por defecto false.

    		String— none

    Output

    'border y border-radius' De un color u otro dependiendo del tipo de la imagen y con borde redondo si es un avatar.
